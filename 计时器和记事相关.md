# 计时器和记事相关

## 创建流程

1. 隐形
 创建 
 后端 给  planID   对应的 records
 后端 给  eventID  对应的系统事件记录
2. 非隐形
 创建 
 后端 给  事件ID  对应的系统事件记录
3. 同步接口
 * 上传  添加字段 eventID events {plan  record}
 * 下载  添加字段 eventID events {plan  record}
4. 系统事件不能删除（素素说）
5. 修改是在对应的设置页面修改（Android提的)


## 数据库建表相关

6. plan 表新增属性 field
 	* type: Int
 	* subtype: Int
 	* end_date: Date

7. event 表
 - plan_id: Int
 - event_id : String UUID 
 - date: date
 - title: String (文字长度)
 - type: Int
  + 系统 0
  + 手动添加 1
  + 自动添加 2
 - notify_time: String （不提醒 是空字符串）(Android)
 - is_delete: bool
